# Vanguard HSE Solutions Website

[![GitHub deployments](https://shields.io)](https://github.io)

Vanguard HSE Solutions (Pty) Ltd is a purpose-driven health and safety solutions company committed to transforming workplace safety across Africa. This repository contains the source code for the company's official multi-page website, built with a culturally aligned, technology-enabled, and human-centred design approach.

---

## 🚀 Live Preview
The website is automatically deployed using GitHub Pages. You can view the live website here:
👉 **[Vanguard HSE Solutions Live Site](https://github.io](https://phodzow0l7.github.io/VanguardHSE/))**

---

## 🌟 Core Pillars

- **Culturally Intelligent:** Reflecting workforce diversity by addressing language, literacy, and socio-economic realities to build ownership from the ground up.
- **Technologically Advanced:** Utilizing AI-powered risk analysis, VR hazard simulations, and data-driven dashboards to modernize safety systems.
- **Human-Centred:** Embedding dignity and ethics into every safety system to ensure sustainable and inclusive workplace standards.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for accessible web layouts (65.5%)
- **CSS3:** Custom styling and responsive layouts (33.5%)
- **JavaScript:** Client-side interactivity and features (1.0%)

VanguardHSE/

## 🔧 Local Development

To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate into the project folder:
   ```bash
   cd VanguardHSE
   ```
3. Open `index.html` in your preferred web browser, or run it using a local development server like **Live Server** in VS Code.
